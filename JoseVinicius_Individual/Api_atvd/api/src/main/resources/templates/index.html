Bem vindo ao Android Individual-API

Rotas da API (http://localhost:8080):

/ - Request - Página inicial

Usuário

  • /login - POST - Envia nome e senha e recebe usuário ou null
  • /usuarios - GET - Usuários do sistema
  • /usuario - POST - adicionar usuário
  • /usuario/{id} - GET - Usuário por id
  • /usuario/{id}/nome/{nome} - PUT - Atualiza nome
  • /usuario/{id}/email/{email} - PUT - Atualiza email
  • /usuario/{id}/senha/{senha}- PUT - Atualiza senha
  • /usuario/{id} - DELETE - Apaga o usuário por id
